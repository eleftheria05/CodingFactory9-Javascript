# CodingFactory9 JavaScript Basics

A simple introductory JavaScript project containing basic examples for learning core JavaScript syntax and concepts.

## 📚 Project Overview

This repository includes beginner-friendly JavaScript examples such as:

* Printing output to the console
* Variable declarations with `let` and `const`
* Increment operators
* Basic comments and code structure

The project is intended for educational purposes and serves as a starting point for learning JavaScript fundamentals.

---

## 🛠 Requirements

* Node.js
* Any code editor (VS Code recommended)

---

## 🎯 Learning Goals

By studying this project, you will learn:

* Basic JavaScript syntax
* Variables and constants
* Console logging
* Simple program execution with Node.js
